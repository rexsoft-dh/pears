# PeaRS

Commnad line  for PRS (Polygenic Risk Score)

## INSTALLATION

- From PIP

```
pip install pears
```

## USAGE

### Preprocessing of cel files


```
pears --preproc --cel 
```

### Imputatation
```
pears --imput
```

### PRS calculation
```
pears --calprs --geno
```

### Annotation
```
pears --annot
```
